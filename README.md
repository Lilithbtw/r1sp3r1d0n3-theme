# R1sp3r1d0n3 Theme
Minimalist theme based on my own website with dracula-like color palette.

## Quickstart
First run: 
```bash
git clone https://github.com/Lilithbtw/r1sp3r1d0n3-theme.git
```
```bash
cd r1sp3r1d0n3-theme
mkdir -p static/fonts
```

Select the font you like the most for the website and copy it to the `static/fonts` folder you created earlier as font.ttf

and finally run To display the markdowns of the `content` folder.
```bash
hugo serve -D 
```

To create the markdown files i recommend using obsidian a github extension to push it to a remote repo.

**Don't forget to change the hugo.toml file**
